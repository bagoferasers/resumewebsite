D.I.A.N.N.A.

Inspiration : To create a deeper passion and inspire the Artemis Generation to push the boundaries of humanity. What it does : It is a trivia survival space game that teaches middle school age kids about space science and diversifies access to educational resources by using a video game interface to drive learning. How we built it : Utilized a game engine python called Pygame, VS code, Github, Git. Challenges we ran into : Learning python and github for the first time, using CSV files adding random characters, tying in multimodal educational resources, using multiple pieces of code from multiple people, learning each other's strengths and weaknesses on the fly. Accomplishments that we're proud of : Making a successful video game. We made a tool to help kids learn more about space and get them excited about the Artemis missions going forward. What we learned : Game programming, Python, collaboration with multiple programmers on the same programmers using Git and Github, using sprites.
