D.I.A.N.A.

Dodging In-game Asteroids Near Artemis ( D.I.A.N.A. ) is a free educational video game to help those with different learning needs to feel more comfortable in their learning environment. It is a trivia survival space game that teaches middle school age kids about space science and diversifies access to educational resources by using a video game interface to drive learning. We utilized a game engine python called Pygame, VS Code, Github, and Git. This was my first time using Python and was a great introduction to the language as well as working in a programming team with different backgrounds. We ended up winning the best game award.
